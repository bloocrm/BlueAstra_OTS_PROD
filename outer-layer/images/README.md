# Features showcase images

Drop screenshots/videos here (or replace the `src` in `Features.html` with your
Supabase public URLs). Each file is optional — if missing, a labelled placeholder
shows instead.

Expected filenames (referenced by Features.html):

| File | Feature |
|------|---------|
| client-dashboard.png    | Client Dashboard (client maintenance, KPIs, revenue) |
| client-view.png         | Client View & List (360° profile) |
| ai-insights.png         | AI Insights |
| employee-dashboard.png  | Employee Dashboard |
| team-access.png         | Team Management & Access Control |
| vendor-management.png   | Vendor Management |
| proposals.png           | Proposals (RFI/RFQ/RFP) |
| communications.png      | Communications Hub |
| ../videos/bloocrm-walkthrough.mp4 | Product walkthrough video |

## Using Supabase instead of local files

Replace each `src` in `Features.html`, e.g.:

    src="https://<project>.supabase.co/storage/v1/object/public/bloocrm/client-dashboard.png"

Recommended image size: ~1280×800 (16:10). PNG or JPG.
