Upload Instructions

Supported Formats: CSV (.csv), Excel (.xlsx, .xls), LibreOffice Calc (.ods), and PDF (.pdf)

File Size Limit: Maximum 5MB per file

Important:

  • First row must contain column headers
  • Use exact column names as specified in the templates below
  • All required fields must be populated
  • Duplicate entries will be skipped
  • Data will be validated before import

Client Data

Best for: Migrating from another system or adding multiple clients at once.

Required Fields: Name, Email, Phone, Business Type, Industry

Best for: Adding bulk leads from marketing campaigns or databases.
